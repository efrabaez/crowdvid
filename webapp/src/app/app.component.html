¿Qué es el Crowdsensing?

El crowdsensing es un paradigma para utilizar diversos dispositivos (como sensores, teléfonos y relojes inteligentes, etc.) para recolectar, almacenar y transmitir información local para el análisis, monitoreo y procesamiento de datos en la toma de decisiones. Dentro de los problemas que solventa este paradigma son sistemas relacionados con planeación de tráfico, monitoreo ambiente, seguridad y recomendaciones sociales.
